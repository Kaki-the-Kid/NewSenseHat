32 // Axis rotation defs
33 //
34 // These allow the IMU to be virtually repositioned if it is in a non-standard configuration
35 // Standard configuration is X pointing at north, Y pointing east and Z pointing down
36 // with the IMU horizontal. There are 24 different possible orientations as defined
37 // below. Setting the axis rotation code to non-zero values performs the repositioning.
